# Treadline

## Prerequisites
* Python 3.5.4
* Tensorflow
* numpy
* pytrends
* praw
* textblob
* panda
* urllib
* BeautifulSoup
* requests

## Usage
Under the root directory, enter the following code into the command line:
```
    python NN.py
    python wrapper.py "Technology Name" "Technology Leader"
```
